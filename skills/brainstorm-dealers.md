# 农资抖音选题生成（面向经销商）

## 背景

内容发布方是一家农资平台，旗下有传媒公司，专注农业内容。内容创作者是平台业务员，熟悉合作模式和商业逻辑，但不具备农技专业知识。可以在农业现场拍摄作为视觉背景，但内容本身只能围绕业务、合作、经营展开，不涉及具体农事操作。

## 平台核心优势

- **不串货**：线上买券线下提货，平台永远不直接接触经销商的客户
- **货源稳定**：自有生产 + 合作厂商双重保障
- **让利空间大**：利润空间远超行业平均（具体数字不在视频中透露）
- **传媒赋能**：帮经销商做短视频脚本、剪辑、拍摄指导、内容培训；平台自己的账号也能给经销商引流
- **产品有实证**：所有产品在经销商处做田间实验，专业视频记录，效果有据可查
- **农业底色**：传媒公司专注农业，内容准确，不会乱说

## 目标受众

- 农资经销商（农药、化肥销售商）
- 覆盖区域：江浙沪、安徽、河南
- 痛点：货源不稳/串货、利润被压缩、不懂如何用短视频开拓客户、担心平台抢客户

## 内容类型说明

### 模式介绍
解释平台的合作模式和商业逻辑，尤其是"不串货"机制、引流赋能方式，打消经销商的顾虑。

### 合作案例
讲述某个经销商通过平台传媒服务或产品合作获得实际收益的真实故事，增强可信度。

### 经销商干货
帮农资经销商解决经营问题：如何用短视频拓客、如何开发大户、如何应对同行竞争等。

### 产品实证
介绍田间实验的过程和结果，展示产品效果有真实数据和视频为证，不是口说无凭。

### 行业观点
对农资行业现状、经销商未来发展方向表达明确立场，帮经销商看清形势。

## 输出格式

严格按照以下格式输出，不要增减字段，不要改变标记符号：

```
【选题1】
选题名称：（10字以内，概括这个选题的核心角度）
角度：（一句话说清楚这个视频的切入点）
为什么对经销商有吸引力：（击中经销商哪个具体痛点或关切）
内容要点：
- （要点1）
- （要点2）
- （要点3，可选）
标题建议：
- （标题1）
- （标题2）

【选题2】
按钮标签：...
（同上）

【选题3】
按钮标签：...
（同上）
```

## 选题质量标准

**避免：**
- 涉及具体农技操作的内容（用药量、施肥节点等业务员不懂的内容）
- 夸大承诺（如"保证赚钱"）
- 泛泛的合作介绍（如"我们平台有多好"）
- 标题党

**好选题的特征：**
- 直接回应经销商的顾虑或痛点
- 有具体场景或对比（如：和普通平台合作 vs 和我们合作的区别）
- 像同行在说话，不像广告
- 利润、客户、风险是核心关键词
